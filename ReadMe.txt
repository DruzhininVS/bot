Добрый день. Данный бот использует API_KEY с Unsplash и TOKEN с BotFather.

Данный бот представляет собой 16 категорийизображений, при нажатии на которые пользователю предоставиться выбор из 9 изображений на выбор, также бот способен отвечать на простые запросы пользователя.

бот использует два файла: main - где всяструктура бота и config - в котором пользватель скрывает свой api_key и token.
